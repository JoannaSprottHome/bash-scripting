Install: https://github.com/nvm-sh/nvm#installation-and-update

nvm ls-remote       - all versions available
nvm ls              - all the versions you have installed
nvm install node    - install the latest version
nvm install 8.15.1  - install specific version
nvm use node        - use latest version
nvm use 11.10.0     - use specific version
nvm run node        - run latest version in node command line (allows for multiple terminals with different versions)
nvm run 11.10.0     - run specific version in node command line (allows for multiple terminals with different versions)
